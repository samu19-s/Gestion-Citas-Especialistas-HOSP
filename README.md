# Sistema de Gestión de Citas Médicas Especializadas (HOSP)

Este repositorio contiene la planificación, gestión de requisitos y documentación inicial para el desarrollo de una plataforma hospitalaria orientada a la gestión de citas en especialidades médicas (Cardiología, Digestivo, etc.).

## 🎯 Objetivo del Proyecto
Optimizar la programación de consultas especialistas para mejorar la atención al paciente y la organización interna del centro hospitalario. El sistema permitirá la solicitud, consulta, reprogramación y cancelación de citas de forma ágil.

## 🛠️ Metodología de Gestión
El proyecto sigue una metodología ágil basada en *SCRUM*. La evolución del mismo se puede seguir a través de las siguientes herramientas de GitHub:
* *Milestones (Hitos):* Divididos en fases de "Gestión del Proyecto" e "Identificación de Requisitos".
* *Issues (Tareas):* Registro de cada actividad, asignación de responsables y seguimiento mediante etiquetas (Labels).
* *Diálogo:* Registro de la toma de decisiones y colaboración síncrona en los comentarios de cada tarea.
